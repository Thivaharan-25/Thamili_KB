# AppLayout

> **Template for:** customer and admin screens.

## Skeleton

- Skip-to-content link.
- Topbar with logo, notifications, theme toggle, user menu.
- Desktop role-aware sidebar.
- Mobile bottom navigation.
- Main landmark wraps routed page.
- Live region announces route changes and async results.

## Rules

- Navigation comes from route/role config.
- Global outage banner sets write actions read-only/disabled.
- Consent banner blocks non-essential telemetry until accepted.
