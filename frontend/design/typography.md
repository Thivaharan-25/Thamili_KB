# Typography

> **Part of:** [[INDEX|T Web Knowledge Base]]  
> **See also:** [[frontend/design/theme|theme]] | [[frontend/design/responsive|responsive]]

| Token | Use | Size |
|---|---|---|
| `text-xs` | metadata, badges | 12px |
| `text-sm` | secondary labels | 14px |
| `text-base` | body and controls | 16px |
| `text-lg` | card headings | 18px |
| `text-xl` | section headings | 20px |
| `text-2xl` | page titles | 24px |
| `text-3xl` | welcome/hero only | 30px |

## Rules

- Do not scale font size with viewport width.
- Letter spacing stays normal.
- Body/form text minimum is 16px.
- Strings live in i18n files.
- Dates use DD/MM/YYYY; currency uses AUD with two decimals unless confirmed otherwise.
